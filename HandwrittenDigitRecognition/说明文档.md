# 手写数字识别
这是一个完成0-9共十个数字的分类问题
## 数据集
数据集下载:直接在torchvision库中下载

## 实验环境
WSL2 Ubuntu-22.04  
conda 26.5.3  
Python 3.14.4  
PyTorch 2.13.0+cu130  
CUDA True(本代码文件未用CUDA)  
numpy 2.5.1  

## 实现思路
### 解析数据 data
数据量较大,不能一次将数据全送入cpu/gpu,这时就需要分批次送入,每一个批次的数据量大小就称为batch_size

### 搭建网络 net
利用全连接线性层搭建一个网络
前向传播、激活函数
这里我们将网络单独定义成一个py文件,需要使用就from  import导入要使用的文件和类即可

### 损失函数 loss
采用交叉熵损失函数

### 优化函数 optimiter
采用Adam优化器,学习率固定为0.01

### 训练模型 training
训练10轮,每轮分为若干batch_size,每个batch_size大小设定为64

### 训练过程中测试模型 test/eval
​仿照训练过程,只不过加载数据从test集中取

### 保存模型 save
pytorch2.6及以上版本对save保存整个模型的安全性做了改进,所以本项目采用保存参数的方式

### 加载模型 load
将保存好的模型加载进来

### 推理 inference
利用训练好的模型进行推理
