# 波士顿房价预测
## 数据集
数据集下载链接：https://archive.ics.uci.edu/ml/machine-learning-databases/housing/housing.data
该数据集打开页面需要ctrl+s保存为housing data纯文本文件。如果是linux系统直接在工作目录下输入命令： wget "上面链接" 即可。

## 实验环境
WSL2 Ubuntu-22.04
conda 26.5.3
Python 3.14.4
PyTorch 2.13.0+cu130
CUDA True(本代码文件未用CUDA)
numpy 2.5.1

## 两个文件
demo_reg是训练文件，保存训练好的模型
demo_regInference是将保存好的模型加载进来，完成推理

## 实现思路
### 解析数据 data
将数据读入，并进行处理，例如去掉多余空格、换行符等操作
随后将数据防止二维列表中方便访问

### 搭建网络 net
利用全连接线性层搭建一个网络
前向传播、激活函数

### 损失函数 loss
采用MSELoss

### 优化函数 optimiter
采用Adam优化器，学习率固定位0.01

### 训练模型 training
训练10000次，观察损失

### 训练过程中测试模型 test
​仿照训练过程，只不过加载数据从test集中取

### 保存模型 save
pytorch2.6及以上版本对save保存整个模型的安全性做了改进，所以本项目采用保存参数的方式

### 加载模型 load
将保存好的模型加载进来

### 推理 inference
利用训练好的模型进行推理
